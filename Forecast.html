Overview

The Forecast class represents weather forecast data for a specific location identified by a ZIP code. It holds information about the current temperature, as well as the day’s future high and low temperatures.

Each temperature attribute is represented using BigDecimal for precision. The class provides a method to convert its data into a hash, allowing for easy serialization or further manipulation.

Examples:

Creating a new Forecast instance

forecast = Forecast.new(zip_code: "90210",
                        current_temperature: BigDecimal("70.0"),
                        day_ahead_high: BigDecimal("75.2"),
                        day_ahead_low: BigDecimal("65.3"))

Converting Forecast instance to a hash

forecast_hash = forecast.to_h
# => { :zip_code => "90210", :current_temperature => BigDecimal("70.0"),
#      :day_ahead_high => BigDecimal("75.2"), :day_ahead_low => BigDecimal("65.3") }
